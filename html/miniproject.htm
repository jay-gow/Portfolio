IOT Programmable Energy Meter for Electrical Load Survey

Mini Project

Abstract

This project is designed to evaluate the consumption of any load in units and cost in rupees based on the energy tariff and the number of hours the load is used. The evaluation is nearly instantaneous, saving valuable time during load surveys. The system uses a microcontroller from the ARDUINO family to calculate the units consumed and the corresponding cost, with user inputs provided via push button switches. The data is then displayed on an LCD. Future enhancements include remote monitoring and load control via a GSM modem.

Objective

The aim of this project is to create a power meter that enables interoperability among multiple meters distributed across a grid. A software-based control panel (running on a server) is developed to communicate with each meter, simulating a smart power grid control center. This system allows the control station to acquire data simultaneously from various meters and even reconfigure or reprogram individual meters remotely.

Project Images

Conclusion

The project demonstrates that real-time monitoring of electrical appliances is achievable through a dedicated website. The processed voltage and current values are displayed on an LCD screen and can be controlled via an application. Simulation using Proteus software confirms that the sensor networks can be effectively deployed in smart grids for continuous monitoring. Additionally, security features such as notifications for load theft further enhance system reliability.

Applications & Advantages

Applications

  • Military applications
  • Power plants
  • Remote energy monitoring and control

Advantages

  • Calibrated and tested sensing module for accuracy
  • Effective deployment in smart grids using wireless sensor networks
  • Reliable and cost-effective solution
  • Reduced manual labor for energy monitoring
